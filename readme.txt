== Description ==

This plugin adds a widget that displays authors who have contributed to the current category and links to their profile page.

If you find this plug-in just so totally awesome that you develop a massive urge to donate, feel free to donate here:
http://wordpressfoundation.org/donate/

All and any feedback - post it on the forum or email wordpress@pointandstare.com

== Installation ==

1. Upload `point-and-stare-caw` folder to your `/wp-content/plugins/` directory  
2. Go to Appearance > Widgets and drag/ drop the `Point and Stare Category Authors` widget to your sidebar
